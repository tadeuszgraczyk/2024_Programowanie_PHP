48a. Zmienne w języku PHP

Zmienne w języku PHP służą do przechowywania danych, które mogą być używane i modyfikowane w trakcie wykonywania skryptu. PHP jest językiem słabo typowanym, co oznacza, że nie musisz deklarować typu zmiennej – typ jest określany automatycznie na podstawie przypisanej wartości.

Deklaracja zmiennych:

W PHP wszystkie zmienne zaczynają się od znaku dolara ($), po którym następuje nazwa zmiennej. Nazwa zmiennej może zawierać litery, cyfry oraz znak podkreślenia (_), ale nie może zaczynać się od cyfry. 

<?php
    $nazwaZmiennej = "wartość";
    $liczba = 42;
    $cena = 19.99;
    $czyAktywny = true;
?>
Zasady nazewnictwa zmiennych:
  • Nazwa zmiennej musi zaczynać się od litery lub znaku podkreślenia (_).
  • Nazwa zmiennej nie może zaczynać się od cyfry.
  • Nazwa zmiennej może zawierać litery, cyfry i znaki podkreślenia.
  • Nazwa zmiennej jest wrażliwa na wielkość liter ($zmienna i $Zmienna to dwie różne zmienne).
Przykłady poprawnych i niepoprawnych nazw zmiennych:
Poprawne nazwy Niepoprawne nazwy
$nazwa $1nazwa
$nazwa_zmiennej $nazwa-zmiennej
$_nazwa $nazwa zmiennej
$nazwa123 $nazwa@zmiennej
Zakres zmiennych:

Zakres zmiennej określa, w jakich częściach skryptu zmienna jest dostępna. W PHP zmienne mogą mieć zakres:

  • Lokalny: Zmienna jest dostępna tylko w obrębie funkcji, w której została zadeklarowana.
  • Globalny: Zmienna jest dostępna w całym skrypcie, poza funkcjami.
  • Statyczny: Zmienna zachowuje swoją wartość pomiędzy wywołaniami funkcji.
<?php
    $globalnaZmienna = "Jestem globalna"; // Zmienna globalna

    function przykladZakresu() {
        global $globalnaZmienna; // Odwołanie do zmiennej globalnej
        echo $globalnaZmienna . "<br>";

        $lokalnaZmienna = "Jestem lokalna"; // Zmienna lokalna
        echo $lokalnaZmienna . "<br>";

        static $statycznaZmienna = 0; // Zmienna statyczna
        $statycznaZmienna++;
        echo "Statyczna zmienna: " . $statycznaZmienna . "<br>";
    }

    przykladZakresu();
    przykladZakresu(); // Statyczna zmienna zachowa swoją wartość
?>
Stałe:

W PHP można również definiować stałe, które w przeciwieństwie do zmiennych, nie mogą być modyfikowane po zadeklarowaniu. Stałe definiuje się za pomocą funkcji define() lub słowa kluczowego const. 

<?php
    define("NAZWA_STALEJ", "Wartość stałej");
    const PI = 3.14;

    echo NAZWA_STALEJ; // Wartość stałej
    echo "<br>PI: " . PI;
?>
Wskazówka: Używaj stałych do przechowywania wartości, które nie powinny być modyfikowane w trakcie wykonywania skryptu, np. konfiguracji aplikacji.
Podsumowanie:

Zmienne są fundamentalnym elementem programowania w PHP. Poprawne używanie zmiennych oraz zrozumienie ich zakresu i typów danych pozwala na efektywne tworzenie dynamicznych aplikacji internetowych.
