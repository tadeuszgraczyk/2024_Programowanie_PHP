49. Typy danych w języku PHP

W języku PHP istnieje wiele wbudowanych typów danych, które umożliwiają przechowywanie różnych rodzajów informacji. PHP jest językiem słabo typowanym, co oznacza, że nie wymaga jawnego deklarowania typu zmiennej. Typ zmiennej jest określany automatycznie na podstawie przypisanej wartości.

Podstawowe typy danych w PHP:

PHP obsługuje następujące typy danych:

Typ danych Opis Przykład
boolean Przechowuje wartości logiczne: true lub false. $czyZalogowany = true;
integer Przechowuje liczby całkowite. $wiek = 25;
float Przechowuje liczby zmiennoprzecinkowe. $cena = 19.99;
string Przechowuje łańcuchy znaków. $imie = "Anna";
array Przechowuje kolekcje wartości w postaci tablic. $owoce = array("jabłko", "banan");
object Przechowuje instancje klas. $osoba = new Osoba();
NULL Reprezentuje brak wartości. $zmienna = NULL;
resource Przechowuje odniesienia do zasobów zewnętrznych, np. połączenia z bazą danych. $polaczenie = mysqli_connect(...);
Przykłady użycia typów danych:
<?php
    // Boolean
    $czyAktywny = true;
    var_dump($czyAktywny); // bool(true)

    // Integer
    $liczba = 42;
    var_dump($liczba); // int(42)

    // Float
    $cena = 12.50;
    var_dump($cena); // float(12.5)

    // String
    $nazwa = "PHP";
    var_dump($nazwa); // string(3) "PHP"

    // Array
    $kolory = array("czerwony", "zielony", "niebieski");
    var_dump($kolory);
    /* array(3) {
      [0]=>
      string(7) "czerwony"
      [1]=>
      string(7) "zielony"
      [2]=>
      string(7) "niebieski"
    } */

    // Object
    class Osoba {
        public $imie;
        public function __construct($imie) {
            $this->imie = $imie;
        }
    }
    $osoba = new Osoba("Jan");
    var_dump($osoba);
    /* object(Osoba)#1 (1) {
      ["imie"]=>
      string(3) "Jan"
    } */

    // NULL
    $brakujacaWartosc = NULL;
    var_dump($brakujacaWartosc); // NULL
?>
Konwersja typów:

PHP umożliwia automatyczną konwersję typów (tzw. type juggling), ale można również wymusić konwersję jawną za pomocą funkcji takich jak intval(), floatval(), strval() lub poprzez rzutowanie. 

<?php
    $liczbaTekst = "123";
    $liczba = (int)$liczbaTekst; // Rzutowanie na integer
    var_dump($liczba); // int(123)

    $tekst = "123.45";
    $float = floatval($tekst); // Konwersja na float
    var_dump($float); // float(123.45)
?>
Sprawdzanie typu zmiennej:

Aby sprawdzić typ zmiennej, można użyć funkcji gettype() lub var_dump(). 

<?php
    $zmienna = 3.14;
    echo gettype($zmienna); // double (w nowszych wersjach PHP: float)
    var_dump($zmienna); // float(3.14)
?>
Wskazówka: Używaj funkcji var_dump() do debugowania, aby zobaczyć typ i wartość zmiennej.
Podsumowanie:

Zrozumienie typów danych w PHP jest kluczowe dla efektywnego programowania. Dzięki elastyczności PHP, programiści mogą łatwo manipulować danymi i dostosowywać je do swoich potrzeb.
