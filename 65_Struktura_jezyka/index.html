48. Struktura i składnia języka PHP

Składnia języka PHP jest stosunkowo prosta i intuicyjna, co czyni go łatwym do nauki dla początkujących programistów. PHP jest językiem skryptowym, co oznacza, że kod jest wykonywany liniowo, od góry do dołu. Pliki PHP mogą zawierać zarówno kod PHP, jak i kod HTML, co umożliwia łatwe osadzanie dynamicznych elementów w statycznych stronach internetowych.

Podstawowa struktura pliku PHP:

Każdy plik PHP powinien rozpoczynać się od znacznika otwierającego <?php i kończyć znacznikiem zamykającym ?>. Znacznik zamykający może być pominięty, jeśli plik zawiera wyłącznie kod PHP. 

<?php
    // Tutaj umieszczasz kod PHP
    echo "Witaj, świecie!";
?>
Podstawowe elementy składni:
  • Instrukcje: Każda instrukcja w PHP kończy się średnikiem (;).
  • Komentarze: PHP obsługuje komentarze jednowierszowe (// lub #) oraz wielowierszowe (/* ... */).
  • Zmienne: Zmienne w PHP zaczynają się od znaku dolara ($), np. $nazwaZmiennej.
  • Typy danych: PHP obsługuje różne typy danych, takie jak liczby całkowite, zmiennoprzecinkowe, łańcuchy znaków, tablice, obiekty, wartości logiczne i inne.
Przykłady podstawowej składni:
<?php
    // Deklaracja zmiennych
    $imie = "Jan";
    $wiek = 25;
    $cena = 19.99;

    // Wyświetlanie wartości zmiennych
    echo "Imię: " . $imie . "<br>";
    echo "Wiek: " . $wiek . "<br>";
    echo "Cena: " . $cena . " zł<br>";

    // Tablice
    $owoce = array("jabłko", "banan", "pomarańcza");
    echo "Mój ulubiony owoc to: " . $owoce[0];

    // Instrukcje warunkowe
    if ($wiek >= 18) {
        echo "<br>Jesteś pełnoletni.";
    } else {
        echo "<br>Jesteś niepełnoletni.";
    }
?>
Struktury kontrolne:

PHP oferuje różne struktury kontrolne, które umożliwiają sterowanie przepływem programu:

  • Instrukcje warunkowe: if, elseif, else, switch.
  • Pętle: for, while, do-while, foreach.
<?php
    // Pętla for
    for ($i = 0; $i < 5; $i++) {
        echo "Iteracja nr: " . $i . "<br>";
    }

    // Pętla foreach
    $kolory = array("czerwony", "zielony", "niebieski");
    foreach ($kolory as $kolor) {
        echo "Kolor: " . $kolor . "<br>";
    }
?>
Funkcje:

Funkcje w PHP pozwalają na grupowanie kodu w celu jego wielokrotnego użycia. Funkcje mogą przyjmować argumenty i zwracać wartości. 

<?php
    // Definicja funkcji
    function powitanie($imie) {
        return "Witaj, " . $imie . "!";
    }

    // Wywołanie funkcji
    $komunikat = powitanie("Anna");
    echo $komunikat;
?>
Wskazówka: Zawsze staraj się pisać czytelny i dobrze sformatowany kod. Używaj wcięć i komentarzy, aby ułatwić zrozumienie kodu innym programistom.
Podsumowanie:

Składnia PHP jest elastyczna i pozwala na szybkie tworzenie dynamicznych stron internetowych. Dzięki prostocie i bogatej dokumentacji, PHP jest doskonałym wyborem zarówno dla początkujących, jak i zaawansowanych programistów.
